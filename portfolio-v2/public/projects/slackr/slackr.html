Slackr

Study Group Application for Students

A comprehensive Android application designed to facilitate online connections among college students through class-based group chats. Built with modern mobile technologies and powered by Google Firebase.

View on GitHub

Built With

Kotlin
React Native
Firebase
Android SDK

Key Features

User Authentication

Secure user authentication powered by Google Firebase, ensuring safe and reliable access to the platform.

Class-Based Groups

Create and join study groups organized by classes, making it easy to connect with classmates.

Real-time Chat

Engage in real-time conversations with group members to collaborate and share knowledge instantly.

Resource Sharing

Share study materials, notes, and resources seamlessly within your study groups.

Social Features

A social media-like environment designed specifically for academic collaboration.

Cross-Platform

Currently being rewritten in React Native to ensure compatibility across iOS and Android platforms.
